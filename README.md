# TSE-Startup-POC

## Lancement Projet

Lancer l'appli avec `docker compose up -d`

## Utilisation de l'aplication
- 2 utilisateurs pré-créé :
  ```json
{
  "username":"root@root.com",
  "password":"rootpass"
}
```

```json
{
  "username":"manager@manager.com",
  "password":"managerpass"
}
```
